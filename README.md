# Work Time Alert

<img src="./public/logo.png" width="200" />

- A simple web application that triggers an audible alert when it's time to start working.

## Requirements

- Golang 1.24.1 or higher  
   - [Download](https://go.dev/dl/)

## Build and Run

```bash	
go build
```

- Linux/Mac
    ```bash	
    ./work-time-alert
    ```
- Windows
    ```bash
    ./work-time-alert.exe
    ```
